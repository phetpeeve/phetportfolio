Tutorial Project

A step-by-step guide with video demonstration

October 2023 Tutorial, Web Development, Guide

About This Tutorial

This tutorial demonstrates how to create a responsive portfolio website from scratch using HTML, CSS, and JavaScript. The video above walks you through the entire process, while the steps below provide a detailed breakdown.

Step-by-Step Guide

1

Project Setup

Create a new folder for your project and set up the basic file structure:

  • index.html (main page)
  • css/style.css (styles)
  • js/script.js (JavaScript)
  • projects/ (folder for project pages)
2

HTML Structure

Set up the basic HTML structure in your index.html file:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Portfolio</title>
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
    <!-- Your content goes here -->
    <script src="js/script.js"></script>
</body>
</html>
3

CSS Styling

Add basic styling to your style.css file. Start with:

/* Reset and base styles */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    color: #333;
}

Then add your custom styles for each section of your portfolio.

4

JavaScript Functionality

Add interactivity to your portfolio with JavaScript. For example, to handle mobile menu:

document.addEventListener('DOMContentLoaded', function() {
    const mobileMenuBtn = document.querySelector('.mobile-menu-btn');
    const navMenu = document.querySelector('nav ul');
    
    mobileMenuBtn.addEventListener('click', function() {
        navMenu.classList.toggle('show');
    });
});
5

Responsive Design

Add media queries to make your portfolio responsive:

@media (max-width: 768px) {
    nav ul {
        display: none;
    }
    
    .mobile-menu-btn {
        display: block;
    }
}
6

Deployment

Choose a hosting service (like GitHub Pages, Netlify, or Vercel) to deploy your portfolio:

  1. Create an account on your chosen platform
  2. Upload your project files
  3. Configure your domain (if needed)
  4. Deploy your site

Conclusion

Congratulations! You've now created a responsive portfolio website. Remember to:

  • Keep your code organized
  • Test on different devices
  • Update your projects regularly
  • Keep learning and improving
